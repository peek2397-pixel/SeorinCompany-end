서린 물류 포털 V30

수정 내용
1. 공지사항
- 공지 관리 권한이 있는 관리자에게 삭제 버튼 표시
- 삭제 전 확인창
- Supabase에서 실제 공지 삭제
- 일반 직원에게는 삭제 버튼 미표시

2. 조직도 수정
- 조직도 수정 클릭 시 중앙에 큰 수정창 표시
- 기존 화면에 빠져 있던 조직 단계·상위 관리자 항목 복구
- 팀 추가, 팀명 수정, 표시순서 변경
- 직원 부서·팀·직급·조직 단계·상위 관리자 수정
- 승인 대상·이동 예정 수정
- 닫기 버튼 및 바깥 영역 클릭 닫기

적용 방법
1. Supabase SQL Editor에서 V30_공지삭제_조직도수정정상화.sql 실행
2. 새 V30 폴더의 index.html 실행
3. Ctrl+F5
4. 공지사항에서 관리자 삭제 버튼 확인
5. 조직도에서 조직도 수정 클릭

중요
- 조직도 저장 기능은 V22_조직도수정_저장안정화.sql의 RPC를 사용합니다.
- V22 SQL을 실행하지 않았다면 먼저 실행해야 합니다.
