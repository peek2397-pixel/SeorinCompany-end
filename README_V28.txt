서린 물류 포털 V25 - 조직도 전체화면 및 팀명 수정

수정 내용
- 조직도가 화면 왼쪽 1/3에만 표시되던 원인 수정
- 기존 .org-chart 3열 Grid를 제거하고 조직도 전체 폭 사용
- PC 화면에서 5개 팀이 가로 전체 너비에 균등 표시
- 조직도 수정 모드에서 각 팀 카드에 '팀명 변경' 버튼 표시
- 팀명 변경 후 해당 팀 직원 소속도 함께 변경
- 기존 조직도 관리창의 팀 추가·순서 변경 기능도 유지

사용 방법
1. V25 압축을 새 폴더에 풉니다.
2. index.html 실행
3. Ctrl+F5
4. 조직도 → 조직도 수정
5. 팀 카드 우측 위 '팀명 변경' 클릭

추가 SQL은 필요하지 않습니다.
단, V22_조직도수정_저장안정화.sql을 실행하지 않았다면 먼저 실행해야 합니다.
