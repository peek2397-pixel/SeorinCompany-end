서린 물류포털 V38 적용 안내

수정 내용
1. 손동오(emp001) 계정을 사원번호 대소문자와 무관하게 구매 최종 승인자로 인식합니다.
2. 최고관리자 표시값만 보지 않고 purchase_final_approve 권한으로 최종 승인 여부를 판단합니다.
3. 김헌정(202305022)은 검토 시작, 검토 완료, 발주 완료, 입고 완료, 구매 완료를 처리합니다.
4. 구매 상태에 맞는 버튼만 표시됩니다.
   신청 → 검토 시작/검토 완료
   검토중 → 검토 완료
   승인대기 → 손동오 승인/반려
   승인완료 → 김헌정 발주 완료
   발주완료 → 김헌정 입고 완료
   입고완료 → 김헌정 구매 완료

적용 순서
1. Supabase SQL Editor에서 V38_구매승인권한_상태버튼_수정.sql 전체 실행
2. ZIP 안의 홈페이지 파일을 기존 파일과 전체 교체
3. 로그아웃 후 emp001로 다시 로그인
4. 구매관리에서 승인대기 건의 승인 버튼 확인

주의
V37 SQL을 이미 실행한 상태를 기준으로 제작되었습니다.
