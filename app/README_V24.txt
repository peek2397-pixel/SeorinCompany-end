서린 물류 포털 V24 - 조직도 화면 맞춤

수정 내용
- 조직도가 화면 왼쪽에 작게 몰리는 문제 수정
- 전체 화면 폭을 모두 사용
- 5개 팀 카드가 화면 폭에 맞춰 균등 배치
- 팀장 카드는 각 팀 최상단
- 인원이 많은 해외팀/B2C는 넓은 화면에서 2열 표시
- 노트북에서는 한 줄 5개
- 화면이 좁으면 3+2 또는 2열로 자동 배치
- 모바일에서는 1열로 자동 전환
- 이름과 직급이 잘리지 않도록 글자 크기와 여백 조정

추가 SQL 실행은 필요하지 않습니다.

적용 방법
1. 압축을 새 폴더에 풉니다.
2. index.html 실행
3. Ctrl+F5
