서린 물류포털 V37 구매관리 적용방법

1. Supabase SQL Editor에서 V37_구매관리_신청검토승인완료.sql 전체를 실행합니다.
2. ZIP 안의 홈페이지 파일 전체를 기존 파일과 교체합니다.
3. EMP001로 로그인해 구매관리 화면을 확인합니다.
4. 김헌정 사원번호 202305022 계정에 구매 승인 권한이 자동 부여되었는지 권한관리에서 확인합니다.

처리 순서
직원 구매 신청 → 김헌정 검토 시작/검토 완료 → 손동오 승인 또는 반려 → 김헌정 발주 완료 → 입고 완료 → 구매 완료

직원은 본인 신청만 조회합니다.
김헌정 구매담당자는 전체 신청을 확인하고 검토·발주·입고·완료 처리합니다.
손동오 최고관리자는 김헌정이 검토 완료한 건만 최종 승인 또는 반려합니다.
