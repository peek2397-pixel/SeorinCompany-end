서린 물류 포털 V16 - 직원별 세부 권한관리

수정 내용
- calendar_manage 컬럼 오류 해결
- 최고관리자는 모든 권한을 유지
- 일반 직원은 필요한 권한만 개별 체크 가능
- 기본 권한 / 전체 해제 / 전체 선택 버튼 추가
- 권한 저장 시 Supabase user_permissions에 영구 저장
- 신규 및 기존 일반직원 권한을 안전한 최소 기본값으로 정리

중요
권한관리 화면에서 EMP001을 선택하면 모든 항목이 체크됩니다.
EMP001은 최고관리자이기 때문에 정상입니다.
김헌정 등 일반 직원을 선택하면 항목별로 자유롭게 체크/해제할 수 있습니다.

적용 방법
1. Supabase SQL Editor에서 V16_권한관리_컬럼보완.sql 전체 실행
2. 새 V16 폴더에서 index.html 실행
3. Ctrl+F5
4. 관리자 로그인
5. 권한관리 → 왼쪽에서 김헌정 선택
6. 필요한 권한만 체크 → 권한 저장

기본 권한
- 대시보드
- 공지 보기
- 조직도
- 일반 소통방
- 1:1 비밀소통
- 근무·휴무 달력
