서린 물류 포털 V13

수정 이유
기존 조직도는 회원가입을 완료한 profiles 직원만 읽어서
EMP001만 표시되고, 가입 전 직원 명부는 표시되지 않았습니다.

V13 변경
- 가입 전 직원도 employee_registry 기준으로 조직도에 즉시 표시
- 가입완료/가입대기 표시
- 제공한 29명 직원 전체 조직도 반영
- 홈페이지 조직도 수정에서 가입 전 직원도 팀·직급·순서 수정 가능
- 가입 후에는 동일 사번 기준으로 profiles 정보와 자동 연결

적용 방법
1. Supabase SQL Editor에서 V13_조직도_직원명부_즉시표시.sql 실행
2. 새 폴더 index.html 실행
3. Ctrl+F5
4. 조직도 메뉴 확인
