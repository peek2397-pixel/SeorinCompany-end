서린 물류 포털 V29 - B2B 원본 로그인 유지

변경 내용
- 업로드한 B2B 작업통계 파일을 수정하지 않고 그대로 사용
- B2B 화면 안의 관리자 로그인 / 직원 로그인 방식 유지
- B2B 원본의 비밀번호와 권한 로직 그대로 유지
- 포털 로그인과 B2B 로그인은 서로 분리
- 포털에서는 B2B 메뉴 접근만 제어
- B2B 화면 안에서는 원본 로그인 권한으로 관리자/직원 화면 구분

사용 흐름
1. 서린 물류 포털 로그인
2. 왼쪽 B2B 작업 통계 메뉴 클릭
3. B2B 원본 로그인 화면에서 다시 로그인
4. 관리자 로그인 시 전체 메뉴
5. 직원 로그인 시 본인 화면만 표시

적용 방법
1. 압축을 새 폴더에 풉니다.
2. index.html을 실행합니다.
3. Ctrl + F5
4. B2B 작업 통계 메뉴를 누릅니다.

추가 SQL은 필요하지 않습니다.
V28의 B2B 권한 컬럼은 그대로 두어도 문제 없습니다.
