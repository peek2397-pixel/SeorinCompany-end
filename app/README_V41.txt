서린컴퍼니 V41 직원 선택형 메신저

추가 기능
- 기존 전체 소통방 유지
- 직원 한 명을 선택하여 1:1 대화 시작
- 직원 여러 명을 선택하여 그룹 대화방 생성
- 그룹방 이름 지정
- 대화방별 최근 메시지와 읽지 않은 메시지 수 표시
- 참여자 목록 확인
- 실시간 메시지 수신
- PC마다 같은 Supabase 데이터를 사용하여 직원 간 연동

적용 순서
1. SQL 폴더의 V41_직원선택형_메신저.sql을 Supabase SQL Editor에서 전체 실행
2. 이 ZIP의 파일로 설치 또는 기존 포털 파일 교체
3. 전 직원 로그아웃 후 재로그인
4. 일반 소통방 메뉴에서 '+ 새 대화' 선택
