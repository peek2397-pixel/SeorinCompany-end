서린 물류 포털 V8

추가 기능
1. 조직도
   - 부서/팀별 직원 카드
   - 재직 직원 수 표시

2. 비상 연락망
   - 본인 연락처
   - 비상연락 대상자
   - 관계
   - 비상연락처
   - 별도 '비상 연락망' 권한을 받은 사람만 조회

3. 일반 소통방
   - 직원 전체 실시간 채팅
   - Enter 전송 / Shift+Enter 줄바꿈
   - 이름, 팀, 직급, 시간 표시
   - 본인 메시지와 다른 직원 메시지 구분
   - Supabase Realtime 연동

적용 순서
1. 압축을 새 폴더에 풉니다.
2. Supabase SQL Editor에서
   V8_조직도_비상연락망_실시간채팅.sql
   내용을 전체 실행합니다.
3. index.html 실행 후 Ctrl+F5
4. 관리자 로그인
5. 직원관리에서 전화번호와 비상연락 정보를 입력
6. 권한관리에서 필요한 직원에게
   - 조직도
   - 비상 연락망
   - 일반 소통방
   권한을 부여합니다.

개인정보 주의
비상연락처는 모든 직원에게 자동 공개하지 않고,
관리자가 '비상 연락망' 권한을 준 직원만 볼 수 있게 만들었습니다.
