서린 물류포털 V40 - 구매물품 사용기간 체크

적용 순서
1. Supabase SQL Editor에서 V40_구매물품_사용기간체크.sql 전체 실행
2. 이 ZIP의 홈페이지 파일로 기존 파일 전체 교체
3. 로그아웃 후 다시 로그인

사용 방법
- 김헌정 구매담당자가 구매관리에서 구매완료 건의 '사용완료' 버튼을 누릅니다.
- 실제 사용완료일과 비고를 입력합니다.
- 포털이 구매완료일부터 사용완료일까지의 기간을 자동 계산합니다.
- 사용 중인 물품은 '사용 중 N일째', 끝난 물품은 '사용완료 N일'로 표시됩니다.

주의
- 구매완료 상태의 건만 사용완료 처리할 수 있습니다.
- 사용완료일은 구매완료일보다 빠르게 입력할 수 없습니다.
