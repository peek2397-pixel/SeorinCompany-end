서린 물류 포털 V14 - 사원번호 전용 로그인

변경 내용
- 직원 화면에서 이메일을 전혀 사용하지 않습니다.
- 회사 사원번호 + 비밀번호로 로그인합니다.
- 회사 사원번호 + 이름 + 비밀번호로 가입합니다.
- 내부적으로만 '사원번호@seorin-portal.com' 계정을 사용합니다.
- 기존 EMP001 관리자 계정은 예전 seorin.local 주소도 자동으로 시도합니다.
- 가입한 직원은 내 정보 메뉴에서 본인 비밀번호를 변경할 수 있습니다.

적용 방법
1. Supabase SQL Editor에서 V14_사원번호전용로그인_확인.sql을 실행합니다.
2. 새 폴더에서 index.html을 실행합니다.
3. Ctrl+F5로 새로고침합니다.
4. 김헌정 직원 가입 시험:
   사원번호 202305022
   이름 김헌정
   비밀번호 6자 이상

Supabase 필수 설정
- Authentication → Email provider ON
- Confirm email OFF
