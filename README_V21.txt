서린 물류 포털 V19 - 카드형 조직도

화면 구조
- 최상단: 물류팀 / 물류총괄 / 총괄 이름
- 중간 연결선
- 하단: 발주팀, 해외팀, 국내팀, B2C, 물류지원팀 등 팀별 카드
- 팀별 인원수 자동 표시
- 팀 안에 직원 이름, 직급, 사원번호, 가입상태 표시

관리자 수정
- 조직도 수정 버튼
- 팀 추가 및 순서 변경
- 직원 팀·직급·표시순서 수정
- 승인 대상 표시
- 이동 예정 표시
- 저장 즉시 조직도 반영

적용
1. Supabase SQL Editor에서 V19_조직도_카드형_표시상태.sql 실행
2. 새 V19 폴더의 index.html 실행
3. Ctrl+F5
4. 조직도·비상연락망 메뉴 확인
