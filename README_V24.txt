서린 물류 포털 V22 - 조직도 수정 저장 안정화

수정 내용
- 조직도 수정 버튼 클릭 시 화면 중앙에 큰 수정창이 바로 열림
- 닫기 버튼 추가
- 직원 팀·직급·순서·상위관리자·표시상태 저장
- 가입 전 직원과 가입 완료 직원 모두 수정 가능
- profiles와 employee_registry 동시 저장
- 팀 추가·수정·삭제 저장 안정화
- RLS 때문에 저장되지 않던 문제를 RPC 방식으로 해결

적용 방법
1. Supabase SQL Editor에서 V22_조직도수정_저장안정화.sql 실행
2. V22 폴더의 index.html 실행
3. Ctrl+F5
4. 조직도 → 조직도 수정
5. 직원 선택 → 수정 → 직원 조직정보 저장
