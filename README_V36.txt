서린 물류 포털 V32 - 직원 본인 달력 / 관리자 전체 달력

동작 방식
- 일반 직원 로그인:
  본인 달력만 표시
  본인 연차·반차·반반차·휴무·주말근무·대체휴일만 등록
  다른 직원 선택 메뉴는 보이지 않음

- 관리자 로그인:
  기본 화면에 전체 직원 일정 표시
  달력 날짜 안에 직원 이름과 일정 구분 표시
  직원 선택으로 한 명만 조회 가능
  특정 직원을 선택하여 대신 일정 등록 가능
  전체 직원 사용내역과 합계 확인 가능

적용 방법
1. Supabase SQL Editor에서 V32_달력_직원본인_관리자전체.sql 실행
2. 새 V32 폴더의 index.html 실행
3. Ctrl+F5

관리자 기준
- 최고관리자
- 또는 권한관리에서 '전체 직원 달력 조회'가 체크된 직원
