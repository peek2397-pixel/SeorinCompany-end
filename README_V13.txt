서린 물류 포털 V10

추가/수정
- 직원 본인 비밀번호 변경
- 연차 부여일수 관리자 입력
- 연차 1일 자동 차감
- 반차 0.5일 자동 차감
- 반반차 0.25일 자동 차감
- 연차 잔여 부족 시 등록 차단
- 주말근무 1일당 대체휴일 1.5일 자동 적립
- 대체휴일 사용 시 잔여 자동 차감
- 대체휴일 부족 시 사용 차단
- 직원 사전 명부 29명 등록
- 직원관리에서 새 직원 추가
- 로컬 하드디스크 매일 자정 Excel/JSON/영수증 자동 백업

적용
1. Supabase SQL Editor에서 V10_연차_반반차_주말근무_직원명부.sql 실행
2. index.html 실행 후 Ctrl+F5
3. 로컬 자동 백업은 LOCAL_BACKUP 폴더 README를 따라 설정
