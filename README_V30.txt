서린 물류 포털 V28 - B2B 로그인 권한 통합

변경 내용
- B2B 작업통계의 기존 관리자/직원 로그인 역할을 포털 로그인에 통합
- 별도 B2B 로그인은 사용하지 않음
- 포털 계정으로 로그인하면 B2B 직원명과 자동 연결
- B2B 직원은 본인 일일작업·상세내역만 표시
- B2B 관리자 권한자는 대시보드·월별평가·반기평가·직원관리·평가기준 전체 표시
- B2B 엑셀 권한자는 엑셀·CSV·백업 메뉴 표시
- B2C KPI 권한과 B2B 권한 완전 분리

권한관리에서 새로 보이는 항목
- B2B 작업통계 보기
- B2B 전체관리·평가
- B2B 엑셀·백업

적용 방법
1. Supabase SQL Editor에서 V28_B2B로그인권한_분리.sql 실행
2. 새 V28 폴더의 index.html 실행
3. Ctrl+F5
4. 권한관리에서 직원별 B2B 권한 설정

권한 예시
- 일반 B2B 직원:
  B2B 작업통계 보기 = 체크
  B2B 전체관리·평가 = 해제
  B2B 엑셀·백업 = 해제

- B2B 관리자:
  세 항목 모두 체크
