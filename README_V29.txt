수정 내용
- B2B 화면이 '포털 로그인 정보를 불러오는 중'에서 멈추던 문제 수정
- 원본 B2B 파일 안의 엑셀 내보내기용 HTML 문자열에 연동 스크립트가 중복 삽입되던 문제 제거
- 포털 로그인 정보를 iframe에 정상 전달
- 최고관리자 또는 KPI 평가 권한 보유자: 모든 B2B 메뉴 표시
- 일반 직원: 개인 일일작업 및 일일상세내역만 표시
- 별도 B2B 로그인과 비밀번호 메뉴 없음

적용
1. 압축을 새 폴더에 풉니다.
2. index.html 실행
3. Ctrl+F5
4. B2B 작업 통계 메뉴 클릭
